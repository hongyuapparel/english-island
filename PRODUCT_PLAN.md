# 英语绘本小馆 · 产品策划

## 定位

**面向 0–3 岁宝宝的英文启蒙绘本 Web App。**
主打母语式输入：家长不用开口，App 帮忙读；支持打印成实体书。
目标用户：英语不自信、想给小宝宝做英语启蒙的家长。

---

## 技术栈（已有）

- Vite + TypeScript + Tailwind CSS，GitHub Pages 部署
- 无后端，数据存 localStorage / IndexedDB
- 绘本数据：`src/data/articles.ts`，`Article` 接口
- AI 水彩插画：Pollinations 免费接口，自动生成并缓存
- TTS 朗读：逐句点读 + 全文连读（已实现）
- 亲子共读模式：段落轮读，橙/蓝双色角色卡（已实现）
- 中文注释开关（已实现）

## 月龄分层（三档）

| 标签 | 月龄 | 宝宝能做什么 | 输入什么 | 篇幅 |
|------|------|-------------|---------|------|
| 🌱 萌芽期 | 0–12个月 | 听、看、感知节奏 | 韵律诗、重复音节、身体部位 | 4–6页 |
| 🌿 探索期 | 12–24个月 | 指物、蹦单词 | 命名绘本、动作词、位置词 | 8–10页 |
| 🌳 故事期 | 24–36个月 | 跟得上情节、认识情绪 | 简单故事、情绪词、社交场景 | 10–14页 |

`Article.category` 命名约定：`'宝宝·0–12月'` / `'宝宝·12–24月'` / `'宝宝·24–36月'`

## 首批 6 本绘本

### 🌱 萌芽期（0–12个月）

**Book 1《Ten Little Fingers》**
- 主题：身体部位，韵律启蒙
- 核心词：fingers / toes / nose / eyes / ears / mouth
- 结构：6页，每页一个部位，押韵句式
- 亲子提示：每页"点一点宝宝的 ___"

**Book 2《Goodnight, Little One》**
- 主题：睡前 routine，安抚情绪
- 核心词：bath / brush / hug / lamp / moon / sleep
- 结构：7页，跟着睡前流程，结尾"I love you"
- 亲子提示：和宝宝一起做每个动作

### 🌿 探索期（12–24个月）

**Book 3《Where Is the Bird?》**
- 主题：位置词，躲猫猫
- 核心词：in / on / under / behind / next to
- 结构：8页，小鸟藏在不同地方，每页一个位置词
- 亲子提示："小鸟在哪里？找到了！"

**Book 4《What Do You See?》**
- 主题：颜色 + 动物命名
- 核心词：red / blue / green / dog / cat / bird / fish
- 结构：9页，每页一种颜色配一只动物
- 亲子提示："你能找到红色的东西吗？"

### 🌳 故事期（24–36个月）

**Book 5《The Hungry Little Bear》**
- 主题：情绪 + 食物，简单情节
- 核心词：hungry / happy / yummy / share / friend
- 结构：10页，小熊饿→找食物→分享→快乐
- 亲子提示：亲子共读，一人读旁白，一人读小熊

**Book 6《Let's Go Outside》**
- 主题：户外探索，自然启蒙
- 核心词：sun / wind / flower / grass / jump / run
- 结构：11页，宝宝和爸爸/妈妈一起出门探索
- 亲子提示：读完真的去指给宝宝看

```ts

  id: 'baby-xxx',

  titleZh: '中文书名',
  category: '宝宝·0–12月',   // 或 12–24月 / 24–36月

  level: 'A1',
  words: 80,
  hook: '一句中文钩子，给家长看的',
  paragraphs: [...],          // 每页一条英文
  translation: [...],         // 对应中文
  vocab: [{ word, meaning }], // 4–6 个核心词
  discussion: [...],          // 亲子互动提示（替代原来的讨论题）
  artNote: '...',             // 插画风格描述，保持全书角色一致
}
```

## 功能优先级

### P0 · 必须有
- 月龄分类书架（全部 / 0–12月 / 12–24月 / 24–36月 四个 Tab）
- 手机绘本阅读（图+文+点读）
- 全书连续朗读
- 单句点读

### P1 · 第一版尽量有
- **PDF 导出/打印**（关键差异点，用 `window.print()` + `@media print` CSS 实现）
- 亲子共读模式（已实现，接上新书即可）
- 中文注释开关（已实现）
- 阅读完成标记（✓ 小仪式感）

### P2 · 之后做
- 收藏/书单
- 阅读天数记录
- 每月上新 1–2 本
- 家长使用指南（每本配一篇"怎么用这本书"）

### 永远不做
- App Store 上架（Web 够用，省 50% 工作量）
- 用户注册/账号系统（localStorage 搞定）
- 测评/练习题（母语式输入不测评，这是产品理念）
- 视频内容
- 社区/UGC
- 0–3 岁阶段的 AI 对话
